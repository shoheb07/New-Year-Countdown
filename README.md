# New-Year-Countdown
New Year Countdown is a dynamic web application built using HTML, CSS, and JavaScript that displays a live countdown timer to January 1st of the upcoming year.
new-year-countdown
│── index.html
│── style.css
│── script.js
Features
Live countdown timer
Automatically detects next year
Updates every second
Displays:
Days
Hours
Minutes
Seconds
Celebration message when countdown ends
Responsive design
Clean and modern UI

🛠 Technologies Used
HTML5
CSS3
JavaScript (ES6)
